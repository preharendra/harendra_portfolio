Loan Default Prediction Dashboard

Problem Statement

Financial institutions needed a systematic way to identify loan default patterns and borrower risk profiles to improve credit decision-making and reduce default rates.

Data Source & Scale

148,671+ loan records encompassing approval patterns, borrower demographics, loan purposes, regional data, and repayment behavior.

Methodology

  • Performed extensive data cleaning including null handling, feature selection, and data transformation to ensure dataset reliability.
  • Built an interactive Power BI dashboard with 10+ KPIs and custom DAX measures for dynamic risk analysis.
  • Applied segmentation analysis to identify high-risk borrower groups by loan purpose, geography, and demographic factors.

Customer Churn Prediction & RFM Segmentation

Problem Statement

A retail business needed to identify at-risk customers and segment its customer base to enable targeted retention strategies and optimize lifecycle marketing investments.

Data Source & Scale

1,000,000+ retail transaction records covering customer purchase history, frequencies, monetary values, and behavioral patterns.

Methodology

  • Engineered customer-level features including Recency, Frequency, Monetary value, AOV, purchase intervals, and return rates.
  • Built an RFM-based segmentation model classifying customers into 5 actionable segments: Champions, Loyal, At-Risk, Lost, and New.
  • Developed and compared Logistic Regression and Random Forest models for churn prediction, selecting Random Forest based on superior performance metrics.

Tools Used

Python Pandas Scikit-learn Random Forest RFM Analysis

Impact

  • Uncovered that the top 10% of customers contributed a disproportionate share of total revenue, informing a high-value retention strategy.
  • Enabled targeted lifecycle marketing interventions through 5-segment RFM classification.
  • Recommended Random Forest model for deployment based on superior F1-score and ROC-AUC performance over Logistic Regression.
